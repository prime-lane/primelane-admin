# PrimeLane

Frontend repo for PrimeLane.

## Development

### Clone the repo

```bash
gh repo clone primelane/primelane
```

### Install dependencies

```bash
npm install
```

### Run the app

```bash
npm run dev
```

## Storybook
The design system is available in Storybook at `http://localhost:6006`.

### Run Storybook

```bash
npm run storybook
```
