# PrimeLane

Frontend repo for PrimeLane.

## Development

### Clone the repo

```bash
gh repo clone primelane/primelane
```

### Install dependencies

```bash
npm install
```

### Run the app

## Storybook

The design system is available in Storybook at `http://localhost:6006`.

```bash
npm run dev
```

### Run Storybook

```bash
npm run storybook
```
